# reccap
A couple of simple scripts for desktop region recording using ffmpeg/avconv and [slop]().

# Usage
Start recording:<br />
`reccap-start [output-filename] [framerate=24]`<br />
Stop recording:<br />
`reccap-stop [output-filename]`
